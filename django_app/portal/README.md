        "### Bước 1: Pull dự án về máy\n",
        "\n",
        "Mở Terminal trong Visual Studio Code và chạy:\n",
        "\n",
        "```bash\n",

        "Nếu máy chưa có dự án, sử dụng:\n",

        "```bash\n",
        "git clone <đường-dẫn-repository>\n",
        "```\n",
        "\n",
        "Sau đó mở thư mục dự án vừa tải về.\n",
        "\n",
        "---\n",
        "\n",
        "### Bước 2: Di chuyển đến thư mục chứa `manage.py`\n",
        "\n",
        "```bash\n",
        "cd django_app\n",
        "```\n",
        "\n",
        "> Nếu Terminal đã đứng tại thư mục chứa `manage.py` thì không cần chạy lệnh này.\n",

        "\n",
        "### Bước 3: Tạo môi trường ảo\n",
        "\n",
        "Chỉ cần thực hiện trong lần cài đặt đầu tiên:\n",

        "```bash\n",
        "python -m venv venv\n",
        "```\n",
        "\n",
        "---\n",
        "\n",
        "### Bước 4: Kích hoạt môi trường ảo\n",

        "venv\\Scripts\\Activate.ps1\n",
        "```\n",
        "\n",
        "---\n",
        "\n",
        "### Bước 5: Nâng cấp `pip`\n",
        "\n",
        "```bash\n",

        "```\n",
        "\n",
        "---\n",
        "\n",
        "### Bước 6: Cài đặt các thư viện cần thiết\n",

        "```\n",

        "\n",
